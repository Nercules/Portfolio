Jobfair App

Introduction

Axxes is a software company in Belgium that participates in many job fairs. Since they are a software company, it would be nice if they had an app where students can enter their information, and everything is stored and synchronized in a database.

Struggles

Since I didn't had worked with React Native and Firebase before, we had to become familiar with the technology first. Also, there was tremendously little time to bring this project to fruition, so it was a race against time.

But in the end, we were able to create a cross-platform mobile application with basic functionality.
