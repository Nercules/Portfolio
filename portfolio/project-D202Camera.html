Person Counter

Introduction

Teachers of the Digital Innovation (DI) program would like to know how many of students are in their classroom every day. Sense you are allowed to work hybrid in this program, some students work from home. Therefor the number of students in class can vary.

Solution

**still in progress**

There are 2 possible sollutions:

1. Place the camera in the corner of the room filming the entire room. Livestream the camera's footage to a software that would count the total number of people in the room. These results would be kept on a server for further use. This would be the more elegant solutiuon.

2. This solution is more complicated and will only be used if the first solution doesn't work (most likely because the model can't detect people far away from the camera). Place the camera in an angle filming the doorway. Livestream the camera's footage to a software that will check the position of the found person. Based on these coördinates we can see if the person left or entered the room. meanwhile the results will be saved on a server for further use.

Which solution is used will be documented soon!
